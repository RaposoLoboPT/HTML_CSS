Principais formatações

Negrito

Nesta frase temos uma palavra em negrito utilizando a tag <b> (Não semântica).

Nesta frase temos outra palavra em negrito mas desta vez utilizando a tag <strong> (Semântica)

Italico

Nesta frase temos uma palavra em italico utilizando a tag <i> (Não semântica).

Nesta frase temos uma palavra em italico utilizando a tag <em> (Semântica).

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>.

Texto grade e pequeno

Estamos a criar um texto grande e um texto pequeno.

Texto riscado

Podemos marcar um texto como riscado para indicar que deve ser lido, mas não é relevante. Podemos representar com a tag <del>.

Texto sublinhado

Podemos marcar um texto como sublinhado para dar uma ênfase. podemos representar com a tag <ins> (Semântica).

Também podemos representar o sublinhado com a tag <u> (Não Semântica).

Texto subrescrito

Para inserir coisas do tipo 22 = 4. Pode ser representado com a tag <sup>

Texto subescrito

Para inserir coisas do tipo H2O. Pode ser representado com a tag <sub>
